  • Vocabulary
    • Node
      • Instance of a computer that runs kuberentes
      • It communicates with the master Node.
      • It runs pods
    • Pod
      • Pod runs 1+ containers
      • Exists on a Node
    • Service
      • Handles requests from one node to another
      • Or public requests to master node.
      • Normally a load balancer.
    • Deployment
      • Defines desirned state.
