How to write essays

tags: #writing #paulgraham #thinking #public

  • Write a draft quickly.
  • Remove things that you are not sure about and is not important
    • RTM approach. This means if you say something you are always correct.
  • Go through the essay a lot of times and repeat the above process.
  • Don’t write the essay if there is nothing left.
  • Write essays for yourselves that are not meant for publishing.
  • Writing essay is a great way to clear your thoughts
