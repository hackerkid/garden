Go basic syntax

tags: #public

  • Go programs are organized into packages

    • Go Package is a collection of go files in the same directory that are compiled togather
    • Functions, types, variables etc declared in one source files visible to other source files.
    • Programs start running in package main

  • A repository contains one or more Go Modules.

    • go.md declares module path: Import path prefix of all the packages with the module.
    • Initialize the go module go mod init github.com/hackerkid/hello-go
    • Install the local go module (Should be in the in the same directory as the module)
      • Various ways to install a local go module
        • go install github.com/hackerkid/hello-go
        • go install .
        • go install
      • The command build the program, creates an executable and put it in $HOME/go/bin
        • The directory can be overrided by setting GOPATH and GOBIN
          • if GOPATH is set the binaries are installed to the bin directory of GOPATH
          • If GOBIN is set the binaries are installed to the GOPATH directory
